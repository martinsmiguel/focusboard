# Focus Board

Aplicativo de gerenciamento de foco e produtividade com timer Pomodoro, quadro de tarefas e log de atividades.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
`npm install`

2. Run the app:
`npm run dev`
